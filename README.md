# Typing Speed Tester

A simple Python project that measures typing speed and typing accuracy.

## Features

* Random sentence selection
* Words Per Minute (WPM) calculation
* Accuracy calculation
* Performance feedback
* Score saving using file handling

## Technologies Used

* Python
* time module
* random module
* File handling

## How to Run

1. Download the project.
2. Open the terminal in the project folder.
3. Run:

```bash
python typing_speed_tester.py
```
